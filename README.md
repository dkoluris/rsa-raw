# rsa-raw
Textbook RSA encode &amp; decode

## Overview
In order to decipher, use `RSA.decipher(YOUR_DATA)`. The `YOUR_DATA` should be a `Buffer`. Keep in mind that this is as simple as possible, padding is disabled.
